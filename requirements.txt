# PathUp Project Architecture v2.0

This project is a full-stack application consisting of a Django (API) backend and a Next.js (React) frontend.

## SYSTEM REQUIREMENTS
1. Node.js (v18+ LTS recommended)
   - Download from: https://nodejs.org/
2. Python (v3.10+ recommended)
   - Download from: https://www.python.org/
3. Memurai (Redis for Windows)
   - Download "Developer Edition" from: https://www.memurai.com/
4. Meilisearch (Search Engine)
   - Download Windows executable from: https://github.com/meilisearch/meilisearch/releases/latest
   - Place `meilisearch.exe` in the `pathup_backend/` directory.

## INSTALLATION

### 1. Frontend Dependencies
Open a terminal in the root directory and run:
cd next_frontend && npm install

### 2. Backend Dependencies
Open a terminal in the root directory and run:
cd pathup_backend
pip install -r requirements.txt
pip install celery redis gevent

## HOW TO RUN
The easiest way to start all 6 services (Redis/Memurai, Meilisearch, Django, Celery Worker, Celery Beat, Next.js) is to use the shortcut:

- Double-click the "run website.bat" file in the project root.

## LOCAL URLs
- Frontend: http://localhost:3000 (or as configured in Next.js)
- Backend API: http://localhost:8000
- Django Admin: http://localhost:8000/admin
